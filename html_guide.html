What is HTML?

HyperText Markup Language (HTML) is the standard markup language used to create websites. It provides the basic structure and contents of a webpage using different tags.

The latest version, HTML5, allows for greater flexibility in the creation of different parts of a website, such as articles, navigation bars, headers, footers, and more.

Main

The Main element represents the main content of the document. It should contain the central topic or functionality of the page.

<main>
 <h1>Main Content</h1>
 <p>This is the main section of your webpage.</p>
</main>

Use the Main tag to encapsulate the primary content of your page, helping assistive technologies and search engines understand the main purpose of your document.

Headings

Headings are used to define the structure of a webpage. HTML provides six levels of headings, from <h1> to <h6>. The <h1> tag represents the highest level of heading, while <h6> represents the lowest.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Use these tags to organize your content and create a hierarchy within your webpage.

Paragraphs

Paragraphs are used to structure and separate blocks of text. The <p> tag is used to define paragraphs in HTML.

<p>This is a paragraph.</p>
<p>Another paragraph.</p>

Use the <p> tag to enclose your text and create distinct sections within your content.

Links

Links are used to navigate between web pages. The <a> (anchor) tag is used for creating links in HTML.

<a href="https://www.example.com">Visit Example.com</a>
<a href="page2.html">Go to Page 2</a>

Replace the href attribute with the URL of the page you want to link to. Links can also be used to navigate within the same page or to external websites.

Images

Images can be included in your webpage using the <img> tag. This tag requires the src attribute, which specifies the path to the image file, and the alt attribute, which provides alternative text for accessibility.

<img src="image.jpg" alt="Description of the image">

Ensure to provide a descriptive alt text for your images to make your content accessible to all users, including those with visual impairments.
